Norbert's Game Framework

Welcome to the official ngf website!
This is the platform where you can access the official runtime engine and development kit for ngf.

ngf is a C# game framework designed for Microsoft Windows, for easy 2D application development.
As of right now, there is no official documentation, but efforts are underway to create one. In the meantime, you can refer to "functions.txt" and "test.cs" for insights.

After installing the runtime engine, execute "create association.bat" as an administrator to establish associations for all files with the ".ngf" extension to the runtime application.
